# clash_ubuntu_keyboard_shortcuts
## 描述
使用keyboard_shortcuts启动clash和修改网络配置
## Keyboard_shorcuts设置
默认把所有文件包括配置文件放在`~/.clash/`
```
| 可能会有不同的结构 ~/.clash/
├── cache.db
├── clash
├── clash.sh
├── config.yaml
├── Country.mmdb
└── stopclash.sh
```
### 开启的快捷键设置 
设置开启脚本为`clash.sh`
### 关闭的快捷键设置
设置关闭脚本为`stopclash.sh`
